## 1.4	实现问题求解的过程

***求1~100间所有偶数的和***

```c
#include <stdio.h>
int main()
{
  int sum = 0;
  int i;

  for(i=0;i<=100;i=i+2)
    sum = sum + i;

  printf("%d\n",sum);
  return 0;
}
```
